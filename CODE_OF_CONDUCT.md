# Code of Conduct

## Our pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent.
- Give actionable feedback.
- Focus on ideas and code, not people.

## Unacceptable behavior

- Harassment, hate speech, or personal attacks.
- Trolling, intimidation, or doxxing.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and PRs that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private report to the maintainer via GitHub contact channels.
